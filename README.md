## 簡述底下應用層協定(英文全名與簡單功能說明):
- HTTP vs HTTPs
  - HTTP
    - HyperText Transfer Protocol 超文本傳輸協定
    - 是一種用來傳輸超媒體文件的應用層協定
    - 是全球資訊網的數據通信的基礎。
    - 是一種無狀態協定
    - 藉由 TCP 作為資料的傳輸方式
  - HTTPS
    - HyperText Transfer Protocol Secure 超文本傳輸安全協定
    - S 是 Secure 的意思
    - HTTPS 透過 HTTP 進行通訊
    - 利用 SSL/TLS 來加密封包
    - 在不安全的網路上建立一個安全信道
